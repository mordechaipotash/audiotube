# AudioTube

YouTube audio streaming web app with magic link authentication.

## What It Does

Stream YouTube audio without the video overhead. Built for:
- Podcast listeners
- Music streaming
- Educational content consumption
- Background audio playback

## Features

- **Magic link auth** - passwordless login via email
- **Audio extraction** - YouTube → audio stream
- **Queue management** - build playlists on the fly
- **Responsive design** - works on mobile and desktop

## Stack

```
HTML · JavaScript · Magic Link Auth · YouTube API
```

## Why Audio-Only?

- **Save bandwidth** - audio is 10x smaller than video
- **Save battery** - no video decoding
- **Less distraction** - focus on content, not visuals
- **Background play** - keep listening while multitasking
